Examples
========

This directory contains JSON payloads referenced by the CLI documentation.

- `cli-calc-batch.json`: batch scenarios for `formulas calc`
- `cli-calc-outs.json`: output refs for `formulas calc --outs`
- `cli-calc-renders.json`: render specs for `formulas calc --renders`
- `cli-build-outs.json`: reduced export refs for `formulas build --outs`
- `serve-calculate-request.json`: request body for `POST /calculate`
